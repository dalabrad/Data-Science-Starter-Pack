# 📊 Data Science Starter Pack

A reusable template for data analysis projects.

---

## 🚀 Purpose

This repository is not a project.

It is a **starter template** for any data science or data analytics project.

---

## 📁 Structure

- data/ → raw and processed datasets  
- notebooks/ → analysis notebooks  
- src/ → reusable code modules  
- reports/ → outputs and figures  

---

## ⚙️ Setup (Windows)

Run:

```powershell
.\setup.ps1

This will:

- Create virtual environment
- Install dependencies
- Prepare project environment

--- 

## 🧠 Philosophy

This template enforces:

- Clean project structure
- Reproducibility
- Separation of raw logic and Analysis

---

##🎯 Usage
- Clone repo
- Run setup.ps1
- Start your project in /notebooks
